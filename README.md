# B4ULeave Plugin

## Overview

Plugin B4ULeave wyświetla modalne okno z pytaniem, czy użytkownik chce opuścić stronę przy zdarzeniu `beforeunload`.

## Installation

```sh
pip install platzky-B4ULeave
```

## Usage

```json
"plugins": [
    {
        "name": "B4ULeave",
        "config": {}
    }
]
```
